# Visual Contract

This skill must preserve the established report language already present in:

- `EDA/operacao_reservatorio/apresentacao_reservatorios.html`
- `EDA/operacao_reservatorio/generate_presentation.py`
- `src/generators/html_report_generator.py`

## Stable elements

- Background: light slate or equivalent soft neutral
- Cards: white, rounded, soft border, soft shadow
- Typography:
  - titles in `Bitter`
  - body in `Inter`
- Accent colors:
  - deep navy blue
  - gold divider or highlight accent
- Recurrent micro-structure:
  - icon bubble
  - colored badge
  - strong section title
  - gold divider
  - content body with explanatory blocks

## Content blocks that may change

- top title
- subtitle
- date and metadata
- figures and images
- analytical summaries
- interpretations
- highlighted findings
- section ordering

## Components to preserve when possible

- header with project identity
- metric cards
- figure narrative cards
- highlighted takeaway callout
- clean placeholder state for missing images

## Principle

Visual stability is more important than novelty.

If a report can be produced by adapting an existing file, prefer that over building a new template.
